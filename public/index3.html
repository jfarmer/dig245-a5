Recommendations

  1. Make the search bar receptive to more keywords
  2. Tabs located on the bottom of the website page need to be more visible by having the navbar fit to screen
  3. When customizing shoes, make the button to switch between shoe elements higher on the screen, so the user does not have to scroll as much
  4. Minimize filter bar, so that users do not have to scroll all the way down to see all the possible filters

Test TakeAways

  1. Wordchoice mattered and tasks were described in a way to not provide hints
  2. All tasks were successful
  3. The user had no prior knowledge to using the website
  4. It was done on a google chrome browser
